Game: Tyrian 0.1
Runs on Windows 64-bit machines
All the source code in in the src folder
The other folders contain the game's resources
jogl.jar is the runnable

Credits:

	Jojo's Bizarre Adventure for music and voices
	Youtube for laser gun sound
	
Controls (ingame):
	W to move up
	A to move left
	S to move down
	D to move right
	L to shoot
	Enter to pause/unpause
	Esc to go back
